## Windows

Vérifier l'environnement
```
  flutter doctor
```

Lancer la démo flutter 
```
  flutter run -d chrome
```

Installer Form Validator
```
flutter pub add form_field_validator
```

## Linux

Lancer la démo flutter 
```
  /usr/local/share/flutter/bin/flutter run -d chrome
```
